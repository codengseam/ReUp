# 综合评分公式

> 本文件定义 `resume-evaluator` skill 的综合评分计算方法。8 维加权 + ATS 分 → 总分（0-100）。

## 1. 8 维权重表

| 维度 | 名称 | 权重 |
|------|------|------|
| D1 | 结构完整性 | 10% |
| D2 | 内容量化度 | 15% |
| D3 | STAR 叙事质量 | 15% |
| D4 | 时间线连续性 | 10% |
| D5 | 关键词密度 | 15% |
| D6 | 技能-经历一致性 | 15% |
| D7 | 错别字/格式 | 10% |
| D8 | 差异化亮点 | 10% |
| | **合计** | **100%** |

权重设计依据：
- D2/D3/D5/D6 各 15%：内容质量与匹配度的核心四维，权重最高。
- D1/D4/D7/D8 各 10%：结构与规范性基础项，权重次之。
- D8 差异化亮点虽权重仅 10%，但它是区分"合格"与"优秀"的胜负手，低分（<5）会额外触发 highlight-extractor 编排。

## 2. 诊断分计算（Diagnostic Score）

每维评分 0-10 分（见 `rules/diagnostics-8dim.md`），加权后换算为 0-100：

```
DiagnosticScore = (D1×0.10 + D2×0.15 + D3×0.15 + D4×0.10 + D5×0.15 + D6×0.15 + D7×0.10 + D8×0.10) × 10
```

结果范围：0-100（保留 1 位小数）。

## 3. ATS 分计算（ATS Score）

仅当用户提供 JD 时计算，见 `rules/ats-scoring.md`：
```
ATSScore = 加权覆盖率（考虑放置位置修正，0-100）
```
无 JD 时 ATSScore = null。

## 4. 综合总分（Total Score）

### 4.1 有 JD 场景
```
TotalScore = DiagnosticScore × 0.65 + ATSScore × 0.35
```
设计依据：DiagnosticScore 衡量简历本身质量（占主），ATSScore 衡量与 JD 的契合（占辅）。ATS 权重 35% 而非更高，因为 ATS 仅是初筛门槛，简历本身的质量更决定面试转化。

### 4.2 无 JD 场景
```
TotalScore = DiagnosticScore
```
此时 D5 关键词密度基于通用高频词表评估，不依赖 JD。

## 5. 总分等级分档

| 总分 | 等级 | 含义 |
|------|------|------|
| ≥ 85 | A（优秀） | 简历质量高，可直接投递目标岗位 |
| 75-84 | B（良好） | 基本就绪，有少量优化点 |
| 60-74 | C（合格） | 需针对弱项改写后投递 |
| 45-59 | D（待改进） | 多维短板，需较大改写 |
| < 45 | F（不合格） | 建议重写或重新梳理经历 |

## 6. 计算示例

### 示例 1：有 JD
8 维得分：D1=7, D2=6, D3=5, D4=8, D5=7, D6=6, D7=8, D8=4
```
DiagnosticScore = (7×0.10 + 6×0.15 + 5×0.15 + 8×0.10 + 7×0.15 + 6×0.15 + 8×0.10 + 4×0.10) × 10
                = (0.7 + 0.9 + 0.75 + 0.8 + 1.05 + 0.9 + 0.8 + 0.4) × 10
                = 6.30 × 10 = 63.0
ATSScore = 78.0
TotalScore = 63.0 × 0.65 + 78.0 × 0.35 = 40.95 + 27.30 = 68.25 → 68.3（C 合格）
```
诊断：D8=4 触发 highlight-extractor 编排；D3=5 需 STAR 重写。

### 示例 2：无 JD
8 维得分：D1=8, D2=7, D3=7, D4=8, D5=6, D6=7, D7=9, D8=6
```
DiagnosticScore = (8×0.10 + 7×0.15 + 7×0.15 + 8×0.10 + 6×0.15 + 7×0.15 + 9×0.10 + 6×0.10) × 10
                = (0.8 + 1.05 + 1.05 + 0.8 + 0.9 + 1.05 + 0.9 + 0.6) × 10
                = 7.15 × 10 = 71.5
TotalScore = 71.5（无 ATS）→ C+ 合格
```

## 7. 置信度标注

诊断报告须附带置信度（0-1）：
```
Confidence = min(1, 简历可解析文本字数 / 800)
```
- 简历文本 < 200 字 → Confidence < 0.25，报告标注"信息不足，评分仅供参考"。
- 简历为图片化/扫描件 → Confidence = 0，报告标注"无法解析，需用户提供文本版"。
- 维度 7 错别字 ≥ 5 → Confidence 额外 ×0.8（态度不严谨影响整体可信度）。
