# 生成提示词保留文档

> 本文档保留紫微斗数专栏生成的专家团评估提示词与写作agent提示词模板，作为资料存档，不在专栏展示。

## 一、专家团评估提示词（Phase 0）

### 架构专家评估提示词
角色：学习路径架构专家。任务：评估紫微斗数学习专栏设计方案的(1)学习路径递进性(2)模块划分合理性(3)修心线贯穿性(4)缺失内容检查(5)文章数量充分性。用WebSearch验证判断，返回评分+具体问题+调整建议。

### 内容专家评估提示词
角色：紫微斗数内容专家，精通中州派/三合派/飞星四化派。任务：评估(1)流派覆盖平衡(2)术语准确性基线(3)覆盖度检查(4)参考资料充分性(5)内容深度建议。用WebSearch补充可引用资料，返回评分+术语基线清单+未覆盖项+补充资料+深度建议。

### QA专家评估提示词
角色：内容工程QA专家。任务：评估(1)质检Checklist完备性(2)引用规范可执行性(3)并行生成风险(4)Loop修复可行性(5)可验收标准细化。返回评分+补充项+改进建议+风险规避措施+细化标准。

## 四、Loop修复提示词模板（Phase 5）

### Round 1（单篇孤立问题）
```
你负责修复第NN篇文章的以下问题：
[问题清单]
请仅修改本篇，不动术语表与其他篇。修复后重新自检。
```

### Round 2（跨篇协调问题）
```
你负责修复[术语一致性/流派表述/交叉引用]类跨篇问题，涉及篇目：[NN,MM,...]。
须持术语表/流派基线，跨多篇统一修改。同类问题禁止并行。
```
