# 紫微斗数专栏术语表（共享契约）

> 每术语唯一权威定义。写作时首次出现须引用本表原文。简繁统一用简体。

## 一、基础术语

| 术语 | 定义 |
|---|---|
| 紫微斗数 | 以出生年月日时为输入，通过星曜、宫位、四化等符号构建命盘推断人生的传统术数体系，列"五大神数"之首 |
| 十八飞星 | 紫微斗数前身体系，见于《续道藏》，以十八颗飞星推算，后被紫微斗数吸收发展 |
| 阴阳 | 万物对立统一的两面，阳主刚主动主明，阴主柔主静主暗 |
| 五行 | 金木水火土五种基本元素及其生克关系；金生水、水生木、木生火、火生土、土生金；金克木、木克土、土克水、水克火、火克金 |
| 天干 | 甲乙丙丁戊己庚辛壬癸十天干，配阴阳五行 |
| 地支 | 子丑寅卯辰巳午未申酉戌亥十二地支，配生肖与时辰 |
| 纳音五行 | 天干地支组合后对应的五行属性（如甲子乙丑海中金），用于定五行局 |
| 五虎遁 | 由年干推月干的口诀（甲己之年丙作首），排盘定宫干之基础 |

## 二、排盘术语

| 术语 | 定义 |
|---|---|
| 命盘 | 以十二宫位为框架、安布诸星的完整推算图谱 |
| 命宫 | 排盘核心宫位，由出生月支顺/逆数至生时定位，主一生总体格局与先天禀赋 |
| 身宫 | 与命宫并立的后天宫位，寄于命/财/官/迁/福/夫六宫之一，35岁后渐主导，主后天造化与结局 |
| 命主 | 命宫纳音五行所属星（如水局命主为破军），非命宫本身，辅助论命 |
| 身主 | 身宫纳音五行所属星，非身宫本身，辅助论身后天 |
| 五行局 | 由命宫天干地支纳音决定的水二/木三/金四/土五/火六局，决定紫微星落位与大限起算岁数 |
| 大限（大运） | 每十年一个运限段落，由五行局定起限岁数，阳男阴女顺行、阴男阳女逆行 |
| 安星法 | 将诸星安入命盘各宫位的方法体系，含安命身宫/起紫微/安府系/月系/时系/年干/年支诸星 |
| 定盘 | 核对出生时辰是否准确的过程，时辰不准则全盘皆错，为解盘前置步骤 |
| 闰月处理 | 排盘遇闰月的规则：以当月十五为界，前半月归上月、后半月归下月（中州派惯例） |
| 子时换日 | 子时（23-1点）跨日问题：早子时（0-1点）归当日、晚子时（23-24点）归次日（或统归次日，需声明） |

## 三、宫位术语

| 术语 | 定义 |
|---|---|
| 十二宫 | 命/兄/夫/子/财/疾/迁/交/官/田/福/父十二宫位，构成命盘框架 |
| 三方四正 | 某宫位与其对宫、三合宫（前后120°两宫）合称三方四正，为解盘黄金法则，四宫同看 |
| 对宫 | 十二宫中相隔六位的宫位（如命宫对宫为迁移宫），互相影响 |
| 夹宫 | 某宫前后两宫所夹，夹吉则吉、夹煞则凶（如日月夹命、羊陀夹忌） |
| 邻宫 | 某宫前后相邻的宫位，关系密切 |
| 暗合宫 | 十二宫中地支六合的两宫（子丑/寅亥/卯戌/辰酉/巳申/午未），暗中呼应影响 |
| 六强宫 | 命/财/官/迁/福/夫六宫，主导人生主要面向，又称六内宫 |
| 六弱宫 | 兄/子/疾/交/田/父六宫，辅助论断，又称六外宫 |
| 立太极 | 以某宫为新的命宫（太极点），重新看其三方四正，用于深度论断某领域 |
| 气数位 | 某宫的第九位（官禄宫之官禄），论该宫位的气数强弱 |

## 四、星曜术语

| 术语 | 定义 |
|---|---|
| 十四主星 | 紫微/天机/太阳/武曲/天同/廉贞（紫微星系六颗，逆布）+ 天府/太阴/贪狼/巨门/天相/天梁/七杀/破军（天府星系八颗，顺布） |
| 紫微星系 | 以紫微为首逆时针安布的六颗主星 |
| 天府星系 | 以天府为首顺时针安布的八颗主星 |
| 六吉星 | 左辅/右弼/文昌/文曲/天魁/天钺，主助力贵人 |
| 六煞星 | 擎羊/陀罗/火星/铃星/地空/地劫，主破坏阻碍 |
| 庙旺平陷 | 星曜亮度四档：庙（最强）/旺（强）/利平（中）/陷（弱），亮度决定星曜吉凶发挥程度 |
| 对星 | 互相关联成对的星曜（如紫微↔七杀、左辅↔右弼），须同看 |
| 杂曜 | 红鸾/天喜/天刑/天姚/三台/八座/龙池/凤阁/孤辰/寡宿等辅佐杂曜 |
| 禄存 | 年干系财神星，主财禄，与天马同宫称"禄马交驰" |
| 天马 | 年支系驿马星，主奔波变动，与禄存同宫主异地发财 |

## 五、四化术语

| 术语 | 定义 |
|---|---|
| 四化 | 化禄/化权/化科/化忌四颗由天干引发的化星，为斗数动态推断核心 |
| 化禄 | 主财禄机遇缘起，表顺遂收获 |
| 化权 | 主权力掌控行动，表强势主导 |
| 化科 | 主名声科甲贵人，表光彩声誉 |
| 化忌 | 主执念阻碍是非，表纠结受阻；非全凶，亦主执着与深耕 |
| 十干四化 | 甲廉破武阳、乙机梁紫月、丙同机昌廉、丁阴同机巨、戊贪阴右机、己武贪梁曲、庚阳武阴同、辛巨阳曲昌、壬梁紫左武、癸破巨阴贪（各天干引发的四化星） |
| 生年四化 | 由出生年天干引发的四化，定一生先天格局重心 |
| 来因宫 | 生年天干所在的宫位，为发射生年四化之"因地"，北派因果论根基 |
| 宫干四化 | 由各宫位天干引发的四化，论宫位间互动因果 |
| 自化 | 宫干四化落回本宫，表内耗、泄漏、自我消耗（向心力自化主收敛、离心力自化主发散） |
| 飞化 | 宫干四化飞入他宫，表因果流向、宫位间影响传递 |
| 大限四化 | 大限宫干引发的四化，管十年运限重心 |
| 流年四化 | 流年天干引发的四化，管一年吉凶 |

## 六、流派术语

| 术语 | 定义 |
|---|---|
| 三合派（南派） | 重星曜性情、三方四正、格局论断，用星多达百余颗，以紫云派为其现代发展 |
| 中州派 | 三合派分支发展，由陆斌兆/王亭之传承，创六十星系互涉、借宫安星法 |
| 飞星四化派（北派） | 重宫干飞化、自化飞化、来因宫，以四化动态推断为核心 |
| 钦天四化 | 蔡明宏/李子阳系统，飞星四化派分支，重数理飞星与来因宫 |
| 河洛理数 | 曾国雄系统，飞星派分支，以河洛数理推宫位互动 |
| 六十星系 | 中州派核心，紫微十二基本盘×对宫组合形成六十种星系组合 |
| 借宫安星 | 中州派技法，当某宫无主星时借对宫星曜论断 |
| 三代论 | 紫云派技法，本命/大限/流年三代叠加论断 |
| 太岁入卦法 | 紫云派技法，将他人生年太岁入自己命盘论六亲关系 |
| 小限 | 以生年支起一岁，男顺女逆逐年行一宫的运限法，看内部迹象，与流年太岁（看外部实际）并行 |
| 斗君 | 流年正月起点星，由流年太岁宫起正月逆数至生月，再起子顺数至生时定之 |
| 格局 | 星曜组合形成的特定格局，分吉格（如紫府朝垣/火贪格）与凶格（如刑囚夹印/羊陀夹忌） |
| 禄马交驰 | 禄存与天马同宫或会照，主奔波中得财 |
| 一六共宗 | 宫位与其第六位（疾厄位）的体用关系，论某宫的体质基础 |
