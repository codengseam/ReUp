# 修心线主题（共享契约）

> 用户核心目的：学会紫微斗数当一面镜子，发现自己的不足并加以修改。
> 双线设计：独立修心篇(31-36) + 每篇修心启示小节。
> 每篇修心启示硬性要求：须引用本篇具体星曜/宫位/四化术语 + 含至少1个可操作的自我观察或行为调整练习。

## 一、各模块修心线主题

| 模块 | 修心线主题 | 修心启示方向 |
|---|---|---|
| 一 基础筑基 | 认识工具：紫微斗数是镜子非算命 | 每篇启示：这一基础知识如何帮助我客观认识自己 |
| 二 星曜精解 | 认识禀赋：每颗星曜是一面性格镜子 | 每篇启示：本篇星曜的正面发挥与负面倾向，我如何自我观察 |
| 三 四化飞星 | 认识动态：四化揭示人生起伏与执念 | 每篇启示：化禄化忌等如何照见我的执着与机遇态度 |
| 四 推断技法 | 认识方法：不同流派是不同视角的镜子 | 每篇启示：多元视角如何帮我更全面看自己 |
| 五 实战解盘 | 认识全局：解盘是系统性自我审视 | 每篇启示：该领域分析如何照见我的现实不足 |
| 六 修心应用 | 转化改过：从认知到修养到行动 | 修心篇正文即修心，启示更深一层 |

## 二、"镜子→发现→修改"闭环映射

| 环节 | 篇目 | 内容 |
|---|---|---|
| 镜子（建立认知工具） | 31 | 紫微斗数作为自我认知之镜：命盘是性格地图非命运判决 |
| 发现（照见不足） | 32-33 | 主星性格偏颇的对治方向 + 命盘缺陷(煞忌破格)的认知 |
| 修改（行动转化） | 34-36 | 流年规划趋吉避凶 + 因果心性提升 + 现代改过工具箱 |

## 三、修心启示写作模板

每篇修心启示须含三要素：
1. **照见**：本篇某星曜/宫位/四化如何反映一种性格倾向或人生模式（须点名本篇具体术语）
2. **对治**：这种倾向的正面发挥与负面陷阱，我可能存在的问题
3. **练习**：一个可操作的自我观察或行为调整练习（如"本周记录自己____的时刻，观察是否____"）

修心篇(31-36)正文本身即修心内容，修心启示小节可谈更深一层的转化或留作综合练习。

## 四、防迷信化底线
- 修心启示须强调"知命非认命"，命盘显示倾向而非决定
- 禁用"命中注定/一定会/必然发生/改命"等宿命论措辞
- 用"倾向/可能/宜注意/可调整"等开放性表述
- 修心的落脚点是"自我修养与行为调整"而非"改运化煞"
