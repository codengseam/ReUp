# 04-安星法与十四主星总览

> 安星法是把星曜"安"进宫位的规矩，十四主星则是命盘的性格主色——先懂分类，再谈深度。

## 一、安星法原理

安星法，是将诸星安入命盘各宫位的方法体系。其骨干有三步：安命身宫、起紫微、安天府系，其余辅煞杂曜皆依附此骨架而生。安命身宫已详[第03篇](./03-排盘方法详解.md)，本篇承接讲起紫微诀与安府系。

### 起紫微诀

紫微星是十四主星之首，其落宫是安布全部主星的基准。起紫微诀由五行局与农历生日共同决定：以局数除农历生日，依商与余数查"起紫微表"定紫微所在宫位（自寅起算）。不同局数（水二、木三、金四、土五、火六）对应不同查表，故同日生人因五行局不同，紫微落宫亦异。这正是"五行局决定紫微星落位"的实操含义。

紫微落定后，紫微星系六颗（紫微、天机、太阳、武曲、天同、廉贞）按固定间隔逆时针布列。六星之间并非等距，各星距紫微的宫位数为固定常数，须查安星表，本篇不展开逐星间隔。

### 安天府系

天府星与紫微星在盘中以寅申连线为轴对称：紫微在寅或申时，紫府同宫；其余位置，天府随紫微对称落位。此对称关系是安天府系的总纲。

天府落定后，天府星系八颗（天府、太阴、贪狼、巨门、天相、天梁、七杀、破军）按固定间隔顺时针布列。由此十四主星全部就位，分两大系统：

- **紫微星系**：以紫微为首逆时针安布的六颗主星，主动、主开创、主显。
- **天府星系**：以天府为首顺时针安布的八颗主星，主守、主承载、主藏。

一逆一顺、一开一守，两大系统在盘中互为表里，构成十四主星的基本张力。月系、时系、年干、年支诸辅煞杂曜则在主星骨架上依各自坐标填入（详见[第05篇](./05-辅星煞星与神煞杂曜体系.md)）。

## 二、紫微星系六星总览

紫微星系六星，皆带"主动开创"的底色，但各有侧重。下表列其基本性情，每星深度见模块二[第08篇](../02-星曜精解篇/08-紫微星系.md)。

| 星曜 | 基本性情 | 一句话定位 |
|---|---|---|
| 紫微 | 尊贵、领导、主观强，略孤高 | 帝王之星，主尊贵与主导 |
| 天机 | 聪慧、善谋、多思易变，略神经质 | 谋士之星，主智略与变动 |
| 太阳 | 热情、博爱、操劳求表现，主男性父辈 | 光明之星，主发散与照拂 |
| 武曲 | 刚毅、决断、重财执行，略孤寡 | 财务将星，主财与刚决 |
| 天同 | 温和、悠闲、福厚情绪化，乐享受 | 福德之星，主安逸与情绪 |
| 廉贞 | 刚柔并济、桃花政事、易情绪化 | 囚星次桃花，主情感与公务 |

紫微星系的整体气质偏"显"：或尊贵主导（紫微），或智谋应变（天机），或光照外放（太阳），或刚决执行（武曲），或安逸享福（天同），或情感纠结（廉贞）。六星皆主动向外投射能量，故多主开创、表现、变动之象。

## 三、天府星系八星总览

天府星系八星，皆带"守成承载"的底色。下表列其基本性情，每星深度见模块二[第09篇](../02-星曜精解篇/09-天府星系.md)。

| 星曜 | 基本性情 | 一句话定位 |
|---|---|---|
| 天府 | 稳重、保守、善守成，主财库 | 帝后之星，主守藏与稳当 |
| 太阴 | 柔顺、感性、重阴财，主女性母妻 | 月亮之星，主柔与蓄 |
| 贪狼 | 多才、欲望、应酬桃花，主学艺 | 欲望之星，主才情与交际 |
| 巨门 | 口才、研究、易招是非猜疑 | 暗星，主口舌与探究 |
| 天相 | 正直、辅佐、被动重衣食 | 印星，主辅佐与规矩 |
| 天梁 | 清高、老成、化灾解厄，略孤 | 荫星，主庇荫与化解 |
| 七杀 | 刚毅、冲动、独立求变动 | 将星，主肃杀与突破 |
| 破军 | 破旧立新、冲动耗散，主变动 | 耗星，主破坏与重建 |

天府星系的整体气质偏"藏"：或稳重守成（天府），或柔顺蓄养（太阴），或才情交际（贪狼），或口舌探究（巨门），或辅佐规矩（天相），或荫庇化解（天梁），或肃杀突破（七杀），或破立耗散（破军）。八星多主承载、收敛、转化之象。

## 四、十四主星的整体格局意义

十四主星并非孤立论断，须与宫位、四化、辅煞合参。本节只点出三个整体要点，深度留待后续。

其一，主星分两系各有取向。紫微系主"显与开"，天府系主"藏与守"。一张命盘若紫微系星曜汇聚于强宫，多主外向开创之格；若天府系汇聚，多主内敛守成之格。两系失衡则性格易偏。

其二，主星须看庙旺平陷。星曜亮度四档：庙（最强）、旺（强）、利平（中）、陷（弱），亮度决定星曜吉凶发挥程度（详见模块二星曜亮度篇）。同一颗星，庙旺时正面尽显，落陷时负面抬头。

其三，主星须看落宫与三方四正。命宫主星定底色，但财帛、官禄、迁移等宫的主星共同刻画人生各面；三方四正会照的主星更会相互加色（详见[第06篇](./06-十二宫位总论.md)）。脱离宫位单论主星，易失之片面。

## 修心启示

**照见**：本篇的"紫微星系"与"天府星系"之分，本身就是一面镜子。紫微系主动主开创，天府系主守主承载——这两类底色，其实就对应人身上"想冲想显"与"想稳想藏"两种倾向。你命宫主星属于哪一系，往往就是你最习惯的应对模式：是遇事先动先表态，还是遇事先稳先观望。

**对治**：正面发挥是认清自己的主星底色后顺势而为，开创型的人去冲、守成型的人去守，各得其所；负面陷阱是用主星标签把自己钉死——"我是紫微就该当领导""我是天同就该享福"，把倾向当借口，反而放弃了另一系的平衡。星有庙旺平陷，人也随境起伏，没有一颗星永远在庙旺。

**练习**：若已知自己命宫主星，本周写下它最突出的一种正面特质与一种负面倾向（如天机的"善谋"与"多疑"）。连续五天，每晚记录当天哪一面出现得多、出现在什么情境。若不知主星，则先观察自己整体偏"开创"还是"守成"，同样记录五天。周末看哪一面过度，下一周有意识补另一面。

## 本篇小结

- 安星法骨干三步：安命身宫、起紫微诀、安天府系，辅煞杂曜依附而生。
- 起紫微诀以局数除生日查表定紫微落宫；天府与紫微以寅申为轴对称，紫微在寅申则紫府同宫。
- 紫微星系六星（紫微/天机/太阳/武曲/天同/廉贞）逆布，主动开创、偏"显"。
- 天府星系八星（天府/太阴/贪狼/巨门/天相/天梁/七杀/破军）顺布，主守承载、偏"藏"。
- 主星须结合庙旺平陷与宫位三方四正合参，单论一星易失片面。
