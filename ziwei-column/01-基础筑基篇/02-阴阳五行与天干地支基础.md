# 02-阴阳五行与天干地支基础

> 阴阳五行与干支是紫微斗数的地基，理解了它们，星曜与宫位的性情才有依归。

## 一、阴阳：万物对立统一的两面

阴阳，是万物对立统一的两面，阳主刚主动主明，阴主柔主静主暗。这是中国古人对世界最朴素的二元划分：日为阳、月为阴；男为阳、女为阴；刚为阳、柔为阴。二者并非对立割裂，而是相互依存、彼此转化。

在斗数中，阴阳是分类的第一把尺子。天干分阴阳、地支分阴阳、星曜也分阴阳（如太阳属阳、太阴属阴）。一阴一阳之谓道，命盘里任何一宫、任何一星，都不孤立存在，必有其对立面与之呼应。

阴阳的要点有三：其一，阳中有阴、阴中有阳，纯阳纯阴极少；其二，阴阳可互根互转，盛极而衰、否极泰来；其三，论命看平衡，过刚易折、过柔易靡。

## 二、五行：金木水火土的生克

五行，是金木水火土五种基本元素及其生克关系。需先记住两组关系：

- **相生**：金生水、水生木、木生火、火生土、土生金。生是滋养、助长、促成。
- **相克**：金克木、木克土、土克水、水克火、火克金。克是制约、克制、规范。

生克并非吉凶的绝对标签。"生"过头则溺爱纵容，"克"得当则规范成器。五行更重"中和"：某行过旺需泄、过弱需扶。斗数中星曜与宫位的五行属性，最终都要回到生克关系里看互动。

五行各有特性，便于记忆：

| 五行 | 特性 | 取象 |
|---|---|---|
| 木 | 生长、向上、条达 | 树木伸展 |
| 火 | 炎上、光明、热烈 | 火焰升腾 |
| 土 | 承载、生化、厚重 | 大地载物 |
| 金 | 肃杀、收敛、刚劲 | 金属锋锐 |
| 水 | 润下、流动、阴柔 | 水流下行 |

五行还有"相生相克"之外的"乘侮"关系：某行过旺则反克所不胜之行（木旺反侮金），某行过弱则被所胜之行反乘。这一层留待进阶，初学先掌握生克主干。

## 三、天干：十干与阴阳五行

天干，是甲乙丙丁戊己庚辛壬癸十天干，配阴阳五行。十天干两两一组，分属五行，奇数为阳、偶数为阴：

| 天干 | 五行 | 阴阳 |
|---|---|---|
| 甲 | 木 | 阳 |
| 乙 | 木 | 阴 |
| 丙 | 火 | 阳 |
| 丁 | 火 | 阴 |
| 戊 | 土 | 阳 |
| 己 | 土 | 阴 |
| 庚 | 金 | 阳 |
| 辛 | 金 | 阴 |
| 壬 | 水 | 阳 |
| 癸 | 水 | 阴 |

天干在斗数中至关重要：四化由天干引发，宫干飞化是北派推断核心（详见模块三）。理解"甲属阳木"这类属性，是后续理解十干四化的前提。

## 四、地支：十二地支与时辰生肖

地支，是子丑寅卯辰巳午未申酉戌亥十二地支，配生肖与时辰。地支是命盘十二宫位的底层坐标——十二宫即沿十二地支排列。

| 地支 | 生肖 | 时辰 | 五行 | 月份(农历) |
|---|---|---|---|---|
| 子 | 鼠 | 23-1点 | 水 | 十一月 |
| 丑 | 牛 | 1-3点 | 土 | 十二月 |
| 寅 | 虎 | 3-5点 | 木 | 正月 |
| 卯 | 兔 | 5-7点 | 木 | 二月 |
| 辰 | 龙 | 7-9点 | 土 | 三月 |
| 巳 | 蛇 | 9-11点 | 火 | 四月 |
| 午 | 马 | 11-13点 | 火 | 五月 |
| 未 | 羊 | 13-15点 | 土 | 六月 |
| 申 | 猴 | 15-17点 | 金 | 七月 |
| 酉 | 鸡 | 17-19点 | 金 | 八月 |
| 戌 | 狗 | 19-21点 | 土 | 九月 |
| 亥 | 猪 | 21-23点 | 水 | 十月 |

地支还构成"六合"与"三合"两类关系，直接影响宫位联动：六合为子丑、寅亥、卯戌、辰酉、巳申、午未，对应命盘的暗合宫（详见[第06篇](./06-十二宫位总论.md)）；三合为申子辰（水局）、亥卯未（木局）、寅午戌（火局）、巳酉丑（金局），对应命盘的三方四正。这两组关系是后续解盘反复用到的几何骨架。

## 五、纳音五行：干支组合的五行属性

纳音五行，是天干地支组合后对应的五行属性（如甲子乙丑海中金），用于定五行局。六十甲子两两配对，共得三十种纳音，每种纳音归属金木水火土之一。

纳音的推算源于古律吕，口诀繁复，初学不必死记三十种纳音名，只需知道它的用途：命宫的天干地支组合查得纳音，纳音的五行属性即决定五行局。五行局是排盘的关键参数，决定紫微星落位与大限起算岁数（详见[第03篇](./03-排盘方法详解.md)）。

下表举例若干，便于理解"干支组合→纳音五行"的对应：

| 干支组合 | 纳音 | 五行 |
|---|---|---|
| 甲子、乙丑 | 海中金 | 金 |
| 丙寅、丁卯 | 炉中火 | 火 |
| 戊辰、己巳 | 大林木 | 木 |
| 庚午、辛未 | 路旁土 | 土 |
| 壬申、癸酉 | 剑锋金 | 金 |

## 六、五虎遁：由年干推月干

五虎遁，是由年干推月干的口诀（甲己之年丙作首），排盘定宫干之基础。地支为固定坐标（正月恒为寅），天干随年干而变，五虎遁就是给出"某年正月起何干"的规则。

因正月地支恒为寅，寅的生肖为虎，故名"五虎"。口诀如下：

| 年干 | 正月天干 | 口诀 |
|---|---|---|
| 甲、己 | 丙寅 | 甲己之年丙作首 |
| 乙、庚 | 戊寅 | 乙庚之岁戊为头 |
| 丙、辛 | 庚寅 | 丙辛之岁寻庚上 |
| 丁、壬 | 壬寅 | 丁壬壬寅顺水流 |
| 戊、癸 | 甲寅 | 若问戊癸何处起，甲寅之上好追求 |

用法举例：某人生于丙年正月，由"丙辛之岁寻庚上"知正月为庚寅，二月顺推为辛卯，三月壬辰，依此类推。由此可定命盘每一宫的天干，宫干正是宫干四化与飞星推断的发动源（详见模块三）。

需注意，五虎遁推出的是月干，配合生月地支即得月柱干支；斗数排盘只用月支定命宫位置（详见[第03篇](./03-排盘方法详解.md)），月干则用于填写宫干。本篇只讲口诀原理，不展开排盘实操。

## 修心启示

**照见**：本篇的"阴阳"揭示一个事实——每个人都有刚柔两面，阳主动主刚、阴主静主柔，二者同属自己。而"五行生克"则说明，不同特质之间会相互助长或制约：你性格中"木"式的生长冲劲，可能被"金"式的收敛规范所克，也可能被"水"式的滋养所生。

**对治**：正面发挥是接纳自身的阴阳两面，刚柔并济、动静相宜；负面陷阱是只认同其中一面，把柔的一面压抑下去，或把刚的一面放任过头，造成内耗。五行的智慧在于"中和"——过旺需泄、过弱需扶，性格调节也遵循此理。

**练习**：本周连续五天，每晚花三分钟记录当天最明显的状态——是"阳"（主动出击、主导表达）还是"阴"（静观倾听、内收退守）。五天后看分布，若严重偏向一极，下一周有意识地在偏向的反方向做一次小尝试（如习惯主导者，找一次场合主动倾听）。

## 本篇小结

- 阴阳是万物对立统一的两面，阳主刚动明、阴主柔静暗，论命看平衡。
- 五行生克：生为金生水/水生木/木生火/火生土/土生金，克为金克木/木克土/土克水/水克火/火克金。
- 天干十、地支十二，皆配阴阳五行；地支六合主暗合、三合主三方四正。
- 纳音五行由干支组合查得，用于定命盘五行局。
- 五虎遁以年干推月干（甲己丙作首），为排盘填宫干之基础。
