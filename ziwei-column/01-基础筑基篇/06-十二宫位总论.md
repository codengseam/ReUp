# 06-十二宫位总论：三方四正、对宫夹宫与宫位联动

> 十二宫是命盘的舞台，宫位之间的联动规则——三方四正、对宫、夹宫、暗合——则是解盘的几何骨架。

## 一、十二宫总览

十二宫，是命、兄、夫、子、财、疾、迁、交、官、田、福、父十二宫位，构成命盘框架。从命宫起逆行安布，每一宫主掌人生一个领域。下表列出各宫含义：

| 宫位 | 别称 | 主掌领域 |
|---|---|---|
| 命宫 | 命 | 一生总体格局、先天禀赋、性格主体 |
| 兄弟宫 | 兄 | 兄弟姐妹、平辈、合伙人 |
| 夫妻宫 | 夫 | 配偶、婚姻、感情、异性交往 |
| 子女宫 | 子 | 子女、生育、晚辈、学生 |
| 财帛宫 | 财 | 财运、收入、理财、物质享受 |
| 疾厄宫 | 疾 | 健康、疾病、身体、灾厄 |
| 迁移宫 | 迁 | 外出、旅行、异地、外在表现、变动 |
| 交友宫 | 交（奴仆） | 朋友、部属、交往对象、下属 |
| 官禄宫 | 官（事业） | 事业、工作、学业、志向 |
| 田宅宫 | 田 | 房产、家庭、家运、不动产、居住 |
| 福德宫 | 福 | 精神、福分、兴趣、心性、潜意识 |
| 父母宫 | 父 | 父母、长辈、出身、文书、长上 |

十二宫并非孤立，而是以命宫为枢纽，向外辐射成一张关系网。命宫定性格主体，其余十一宫分掌各领域，彼此通过固定的几何关系联动。理解联动规则，比死记每宫含义更重要。

## 二、三方四正与对宫

### 三方四正

三方四正，是某宫位与其对宫、三合宫（前后 120° 两宫）合称三方四正，为解盘黄金法则，四宫同看。具体而言：本宫加两个三合宫为"三方"（共三宫），再加对宫为"四正"（共四宫）。三合宫由地支三合决定，恒为本宫前后各隔四位的两宫。

以命宫为例，命宫的三方四正如下表：

| 关系 | 宫位 | 说明 |
|---|---|---|
| 本宫 | 命宫 | 主体 |
| 三合宫 | 财帛宫、官禄宫 | 与命宫成 120°，论财力与事业 |
| 对宫 | 迁移宫 | 隔六位相对，论外在与变动 |

故论命宫须"命、财、官、迁"四宫同看：命是主体，财官是支撑主体的两端财力与事业，迁是命的外在投射。脱离三方四正单看一宫，论断易偏。此规则适用于所有宫位：论财帛则看财、福、官、命；论夫妻则看夫、官、福、迁——每一宫都有自己的三方四正组合。

### 对宫

对宫，是十二宫中相隔六位的宫位，互相影响。六组对宫固定不变：

| 对宫组合 | 联动含义 |
|---|---|
| 命 ↔ 迁 | 内在自我与外在表现 |
| 兄 ↔ 交 | 平辈与部属、合伙与下属 |
| 夫 ↔ 官 | 婚姻与事业（互相牵制） |
| 子 ↔ 田 | 子女与家宅、生育与居住 |
| 财 ↔ 福 | 财力与福分、赚钱与享受 |
| 疾 ↔ 父 | 身体与父母、灾厄与出身 |

对宫的关系是"互为表里、互相牵动"。命宫强则迁宫弱可能主内向，迁宫强则多外出表现；财宫与福宫一进一出，失衡则要么只赚不享、要么只享不赚。对宫是三方四正中最直接影响本宫的一环。

## 三、夹宫、邻宫与暗合宫

除三方四正与对宫外，还有三种宫位联动，影响力次之但不可不知。

### 夹宫

夹宫，是某宫前后两宫所夹，夹吉则吉、夹煞则凶。如命宫被其前后两宫（父母宫与兄弟宫）所夹，若两宫吉星云集则命宫受夹助，若两宫煞忌汇聚则命宫受夹制。经典说法如"日月夹命"（太阳太阴夹命宫主贵）、"羊陀夹忌"（擎羊陀罗夹化忌主凶），皆取此理。夹宫看的是本宫的"环境"——左右两宫为本宫提供了一个被包围的气场。

### 邻宫

邻宫，是某宫前后相邻的宫位，关系密切。邻宫之间常互相渗漏：命宫与兄弟宫相邻，性格受平辈影响；夫妻宫与子女宫相邻，婚姻与生育相连。邻宫的影响弱于对宫与三方，但论六亲与连续事件时常用。

### 暗合宫

暗合宫，是十二宫中地支六合的两宫（子丑、寅亥、卯戌、辰酉、巳申、午未），暗中呼应影响。与三方四正的"明会"相对，暗合是"暗助"或"暗耗"。以命在午为例，午六合未，命宫的暗合宫即未宫之父母宫，暗示命与父母有暗中呼应。暗合宫在论隐蔽关系、潜在影响时方显其用，初学先知其理即可。

## 四、身宫与六强六弱宫

### 身宫

身宫，是与命宫并立的后天宫位，寄于命、财、官、迁、福、夫六宫之一，35 岁后渐主导，主后天造化与结局。身宫的安法与寄宫规则已详[第03篇](./03-排盘方法详解.md)：子午时寄命、丑未时寄福、寅申时寄官、卯酉时寄迁、辰戌时寄财、巳亥时寄夫。

命身关系是斗数论命的一对核心张力：命宫主先天格局与性格底色，身宫主后天走向与结局造化。35 岁前命宫权重高，35 岁后身宫渐显。身宫所寄之宫，往往是此人后天最易被牵引、也最需修习的领域——寄财者后天多被财事牵引，寄迁者后天多与外出变动相伴。

### 六强宫与六弱宫

十二宫按权重分强弱两组：

| 分类 | 宫位 | 又称 | 作用 |
|---|---|---|---|
| 六强宫 | 命、财、官、迁、福、夫 | 六内宫 | 主导人生主要面向 |
| 六弱宫 | 兄、子、疾、交、田、父 | 六外宫 | 辅助论断 |

六强宫恰是身宫可寄的六宫，主掌自我、财力、事业、外动、精神、婚姻六大核心领域，是论命重心所在。六弱宫主六亲与健康家宅，作辅助与背景论断。解盘时强宫先看、弱宫辅参，主次分明才不致失焦。

需说明，强弱之分是权重而非吉凶——强宫未必吉、弱宫未必凶，且强弱宫位深度推断（如某宫化忌的具体论法、立太极等技法）留待模块五解盘篇展开，本篇只建立框架。

## 修心启示

**照见**：本篇的"十二宫"把人生切成十二个领域，命宫是主体，其余是不同面向。而"三方四正"提示一个深刻的事实——任何一面都受相关三面牵动，没有哪个领域能独立成立。尤其"对宫"的拉扯，几乎是人生的常态：命与迁（自我与外在表现）、财与福（赚钱与享受）、夫与官（婚姻与事业），人常在对立的两端之间挣扎取舍。

**对治**：正面发挥是用十二宫框架系统审视自己各领域，看出哪面失衡；负面陷阱是只盯命宫一处给自己下定义，忽略了"我"其实是由十二个面向共同构成。身宫"35 岁后渐主导"也提醒：人是会变的，后天的走向能重塑先天的底色，不必把早年性格当作一辈子。

**练习**：本周挑一组对你最纠结的对宫领域（如命-迁的内向与外向、或财-福的攒钱与花钱），连续五天记录两者如何互相拉扯——哪天偏向这一端、哪天偏向那一端、触发情境是什么。周末回看，找一次情境，下次尝试让两端各让一步，看是否更舒服。

## 本篇小结

- 十二宫从命宫起逆行安布，命兄夫子财疾迁交官田福父，各主一个领域。
- 三方四正是解盘黄金法则：本宫+两个三合宫+对宫，四宫同看（如命财官迁）。
- 对宫六对固定：命迁、兄交、夫官、子田、财福、疾父，互为表里互相牵动。
- 夹宫看前后两宫所夹的"环境"，邻宫主相邻渗漏，暗合宫主地支六合的暗中呼应。
- 身宫寄六强宫之一，35 岁后渐主导；六强宫（命财官迁福夫）主导、六弱宫（兄子疾交田父）辅助。
