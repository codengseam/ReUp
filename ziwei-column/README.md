# 紫微斗数学习专栏：以斗数为镜，照见自我，修心改过

> 学会紫微斗数，并非为了算命，而是以命盘为镜——认识自己的性格禀赋、潜能倾向与人生节律，发现不足，加以修改。
> 本专栏共 6 模块 36 篇，从零基础到能独立解盘，贯穿"镜子→发现→修改"的修心主线。

---

## 专栏导览

### 第一性原理

紫微斗数的本质是一套以出生时间为输入、以星曜宫位四化为符号的认识工具。其核心价值不在预测命运，而在提供一套认识自我的符号语言。学会它，你就拥有了一面照见性格特质与人生模式的镜子。

### 学习路径

```
基础筑基 → 星曜精解 → 四化飞星 → 推断技法 → 实战解盘 → 修心应用
(认识工具)  (认识禀赋)  (认识动态)  (认识方法)  (认识全局)  (转化改过)
```

## 使用说明

- **零基础读者**：按 01→36 顺序通读，建立完整知识体系
- **有基础读者**：可按需跳读，模块三(四化)、模块四(流派技法)、模块六(修心)是精华
- **修心导向读者**：先读 31 建立认知框架，再回头学技术，最后读 32-36 落地修心
- 每篇文章末尾的"参考资料"不在专栏展示，仅供深度查阅
- 排盘示例均用文字表格呈现，建议配合在线排盘工具实操

## 修心主线

本专栏的核心目的不是算命，而是修心。命盘是一面镜子，照见你的性格倾向与人生模式：

| 环节 | 篇目 | 核心问题 |
|---|---|---|
| 镜子 | 31 | 命盘是性格地图，不是命运判决 |
| 发现 | 32-33 | 我的主星有哪些偏颇？命盘有哪些缺陷？ |
| 修改 | 34-36 | 如何用流年规划指导决策？如何改过修行？ |

知命非认命。命盘显示的是倾向，不是决定。修心的落脚点在于自我修养与行为调整。
